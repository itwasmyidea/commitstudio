## Prerequisites

Before using CommitStudio, you need to have:

- **Node.js** v18 or higher
- **Git** installed and configured
- A **GitHub account** with access to your repositories
- An **OpenAI API key** for AI-powered analysis

## Installation

Install CommitStudio globally using npm:

```bash
npm install -g commitstudio
```

For detailed installation instructions, see the [Installation Guide](/docs/1-getting-started/installation).

## Basic Usage

Once installed, you can run CommitStudio in your Git repository:

```bash
commitstudio --help
```

This will display the available commands and options.
